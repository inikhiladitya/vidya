[[Network External Storage]]

Storage Area Network uses specialized equipment that deals only with disk [I/O](#I%2FO) over a specialized storage network.

- Hosts access remote network storage as opposed to [DAS](#DAS)
- Host book device can exist on a SAN
- Dedicated network for disk I/O traffic over a network (nothing to do with [TCP/IP](#TCP%2FIP) traffic)
- Fibre Channel (FC)
- Fibre Channel over Ethernet (FCoE)
- iSCSI
