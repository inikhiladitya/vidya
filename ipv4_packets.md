title: IPv4 Packets
---

IPv4 Packets (Packet Header) contain the following information:

- Source Address
  - IP of sender
- Destination Address
  - IP of receiver
- IP Flags
  - Allows packet fragmentation
- Protocol
  - [TCP](#TCP) or [UDP](#UDP)
